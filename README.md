Email OOP Code description

Overview:
The "OOP Email Simulator" is a Python program that simulates email functionality using object-oriented programming. It allows users to interact with email objects within a simulated inbox environment.

Key Features:

    Email Class: Implements essential attributes and behaviors of email communication, including sender's email address, subject line, content, and read status.
    Inbox Management: Utilizes a dedicated list (inbox) to store email objects, enabling efficient storage and retrieval operations.
    User Interaction: Presents users with a menu-driven interface to read emails, view unread emails, or exit the application.
    Main Program Logic: Orchestrates user interactions and ensures smooth program operation through a well-structured while loop and robust error handling mechanisms.
    Functional Simplicity: Offers intuitive user experiences and streamlined functionalities, making email simulation accessible to users of all skill levels.
    Educational Value: Provides hands-on experience in object-oriented design and programming principles, serving as a valuable learning resource for developers.
